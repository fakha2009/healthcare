# 🌿 Healthcare Sanatorium System

Добро пожаловать в десктопное приложение для управления санаторными путёвками и записью пенсионеров. Этот проект демонстрирует современный Windows Forms интерфейс, простую архитектуру и удобный рабочий процесс для операторов медицинских учреждений.

## 🚀 Что это за проект

- Полноценное десктопное приложение на `C#` и `Windows Forms`
- Управление пенсионерами, санаториями, путёвками и отчётами
- Красивый и понятный UI в стиле Glassmorphism
- Лёгкая интеграция с `Microsoft Access (.accdb)` через `OLE DB`
- Быстрая сборка через `Python` и `csc.exe`

## ✨ Преимущества

- **Мгновенный поиск и фильтрация** по ФИО, номеру удостоверения, региону и статусу
- **Полная работа с путёвками**: создание, правка, удаление, печать и экспорт
- **Отчётность в PDF**: выгрузка видимых строк, выбранной записи и сводных таблиц
- **Быстрый демо режим**: очистка демонстрационных данных и восстановление статистики
- **Удобный интерфейс**: светлая/тёмная тема, уведомления, адаптация под 4K

## 🧩 Основные разделы

## 📂 Структура проекта

```text
src/
  HealthcareInterface.cs   # основная логика интерфейса
  HostProgram.cs           # загрузка сборки через Reflection
  Launcher.cs              # выбор x86/x64 и OLE DB провайдера

docs/
```

## ⚡ Быстрый старт

1. Запустите приложение из папки `build/`

```powershell
build\HealthcareSanatoriumInterface.exe
```

2. Если нужно пересобрать, выполните:

```powershell
py build.py
```

## 🎯 Что сделано

### Асинхронная загрузка таблиц
- **Async LoadGrid** для разделов Пенсионеры, Путевки и Санатории
- Открытие раздела больше не замораживает UI
- Статус загрузки отображается в реальном времени: `"Загрузка..."` → `"Записей: 24"`
- Форма остаётся отзывчивой во время работы с БД

### Оптимизированный сервис данных
- **Async RefreshStats** для фоновых SQL-запросов
- 3 запроса к БД выполняются асинхронно, не блокируя кнопки управления
- Мгновенный доступ к инструментам даже при обновлении статистики

### Удалены лишние операции
- Избавились от многозадачного `UiPerformance.Optimize()` при переключении разделов
- Отказались от рекурсивного обхода контролов — сэкономили ресурсы CPU
- Используются конструкторы с `DoubleBuffered = true` для плавной отрисовки

### Сортировка таблиц по клику на заголовок
- Клик на любой заголовок колонки сортирует данные по возрастанию/убыванию
- Клиентская сортировка — никаких дополнительных запросов к БД
- Быстро и удобно

### Горячие клавиши во всех трёх формах

| Клавиша | Действие |
|---------|----------|
| `F5` | Обновить таблицу |
| `Insert` | Добавить новую запись |
| `Delete` | Удалить выбранную запись (если таблица в фокусе) |
| `Enter` в поиске | Мгновенный поиск (без ожидания 350мс) |

**Результат:** интерфейс работает мгновенно, даже на совсем слабых ПК. Никаких фризов, никаких тормозов — просто удобно.
